# Finely Cred — Logo & Brand Assets

All brand files live in this folder. Use these paths in the app, emails, and marketing.

## Quick access (local dev)

When running `npm run dev`, open these URLs:

| Asset | URL |
|-------|-----|
| **Primary logo (SVG)** | http://127.0.0.1:5173/brand/finely-cred-logo.svg |
| **Icon / favicon (SVG)** | http://127.0.0.1:5173/brand/finely-cred-icon.svg |

## Files in this folder

| File | Use |
|------|-----|
| `finely-cred-logo.svg` | Main wordmark — web, print, scalable |
| `finely-cred-icon.svg` | Square icon — favicons, app tiles |

## Referenced in app (add PNG exports when available)

These paths are used by the live site; add the PNG files here when you export from design:

| File | Use |
|------|-----|
| `finely-cred-logo-dark.png` | Dark backgrounds (nav, portal) |
| `finely-cred-logo-light.png` | Light backgrounds |
| `finely-cred-logo-email-dark.png` | HTML emails on dark |
| `finely-cred-logo-email-light.png` | HTML emails on light |
| `finely-cred-logo-email-signature.png` | Email signature block |

## Copy to your desktop

**Windows (PowerShell):**

```powershell
Copy-Item "public\brand\finely-cred-logo.svg" "$env:USERPROFILE\Desktop\finely-cred-logo.svg"
Copy-Item "public\brand\finely-cred-icon.svg" "$env:USERPROFILE\Desktop\finely-cred-icon.svg"
```

**From repo root:** `Tishobe/finely-cred-main/public/brand/`
