# Launch OS — SOP Master

Plain-English playbooks for Finely Cred. Source of truth: `src/domain/platformSops.ts`.  
Live UI: `/help-center` (public) and `/admin/launch-os` (staff + checklist gates).

## Public

### Start here — pick your path
- **When:** Someone lands on Finely and is not sure where to go.
- **Owner:** finely_advisor
1. Open Start Here → `/start-here`
2. Watch the overview → `/resources#videos`
3. Begin intake → `/onboarding`

### Choose a credit monitoring service
- **When:** User needs a tri-merge report source for HTML upload in the portal.
1. Open Resources monitoring section → `/resources#monitoring`
2. Pick HTML-friendly provider (IdentityIQ, SmartCredit, MyScoreIQ)
3. Use official free report if needed → AnnualCreditReport.com

## Portal

### Upload your credit report
1. Go to Reports → `/portal/reports`
2. Upload HTML if possible
3. Review parsed items

### Write and mail a dispute letter
1. Open Disputes → `/portal/disputes`
2. Open Letter Studio → `/portal/letters`
3. Attach evidence → `/portal/documents`
4. Export PDF and mail

### Pay or update billing
1. Open Billing → `/portal/billing`
2. Update payment method
3. Contact support if locked → `/portal/messages`

## Admin

### Create a client file (partner)
1. Open Partner Management → `/admin/partners`
2. Create partner
3. Send claim link

### Upload report for a client
1. Open client profile
2. Reports tab — upload HTML or PDF
3. Verify parse

### Triage today's ops queue
1. Open Ops command center → `/admin/workflow`
2. Handle SLA breaches first
3. CRM pulse → `/admin/crm`

## Affiliate

### Share your referral link
1. Open Affiliate Hub → `/affiliate/hub`
2. Copy tracked link
3. Estimate earnings

### Pitch Denefit contracts
1. Open Denefits tab
2. Explain reporting path (results vary)
3. Message partnership line if unsure

## Agent

### Work a client file as specialist
1. Specialist Hub → `/credit-specialist/hub`
2. Open client dashboard
3. Execute letter workflow → `/portal/letters`

## Business

### Build business vendor stack
1. Business profile → `/business/profile`
2. Vendor stack → `/business/vendors`
3. Lender logic → `/business/lender-logic`

## Compliance

### Compliance check before mailing
1. Evidence gate
2. Language review — factual, educational tone
3. Identity vault when required

---

**Compliance:** Educational dispute workflow only — not legal advice. No guaranteed deletions or score outcomes.

**Tours:** Each SOP links to a manifest tour — preview in Admin → Tour Studio or tap **Watch how** on the live page.
